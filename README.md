# Projeto de Binarização de Imagem

Este projeto realiza a conversão de uma imagem colorida para escala de cinza e binarização, utilizando Python e a biblioteca Pillow.

**Como funciona:**
- `transforma_imagem.py`: Script em Python que transforma a imagem original em duas novas imagens (cinza e binária).
- Basta rodar o script e usar seus próprios arquivos de imagem.

Atividade desenvolvida para fins educacionais.

## Arquivos
- `image.gif` - Imagem original
- `image_gray.gif` - Imagem em tons de cinza
- `image_binary.gif` - Imagem binarizada (preto e branco)
